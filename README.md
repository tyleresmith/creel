## Creel

## Description
A web app for saving and sharing your fishing trips! Record your trips and log your trophy catches

## Install

- Clone repo

- install Gemfile
``` javascript
bundle install
```

- Migrate database
``` javascript
rake db:migrate
```

- install Bowerfile
``` javascript
rake bower:install
```

- Run server
``` javascript
rails s
```
